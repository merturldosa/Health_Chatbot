# 🚀 AI 건강 상담 앱 - Phase 1 개발 로드맵

> **목표**: 정신 건강, 임산부, 만성질환자, 일반인을 위한 통합 건강 관리 웹앱
> **개발 기간**: 3-4개월 (혼자 개발)
> **예산**: 최소 (무료/저비용 API 및 오픈소스 활용)
> **출시 플랫폼**: 웹 (모바일 반응형) → 이후 React Native로 앱 전환

---

## 📋 개발 전략

### 핵심 원칙
1. ✅ **외부 API 최대 활용** - 직접 개발 최소화
2. ✅ **오픈소스 활용** - 검증된 라이브러리 사용
3. ✅ **점진적 확장** - MVP → 기능 추가 방식
4. ✅ **모바일 우선** - 반응형 웹 디자인
5. ✅ **무료 서비스 활용** - 배포 및 인프라 비용 최소화

### 기술 스택 (현재 + 추가)
```
백엔드:
- FastAPI (기존)
- Google Gemini 1.5 Pro (기존)
- PostgreSQL (프로덕션)
- Redis (세션 캐싱)

프론트엔드:
- React + Vite (기존)
- TailwindCSS (UI 라이브러리 추가)
- React Native (Phase 2 - 앱 전환용)

외부 API:
- Spotify API / YouTube API (음악 치료)
- Unsplash API (이미지)
- Web Audio API (바이노럴 비트)
- 건강보험심사평가원 API (병원 정보)

배포:
- Vercel (프론트엔드)
- Render.com (백엔드)
- Supabase (데이터베이스 - 무료)
```
